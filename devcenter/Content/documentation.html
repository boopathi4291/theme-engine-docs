Volusion Theme Engine, Storefront API, and You

Storefronts and company websites have become far more interactive experiences for shoppers, accessed on a variety of devices, mobile phones, and computers. Volusion sees the trends and responds with new opportunities to match your business needs and dreams with the latest technologies.

There are no limits to what you can create with the Theme Engine. Let your imagination run wild and create that site you have been dreaming about. The Theme Engine will include a variety of pre-built starter themes to empower you in creating beautiful websites and custom apps. You can leverage any number of available web components or code and use your own with these designs.

Coming soon, you can add new responsive themes to your storefronts, powered by a Volusion theme engine and supported with a REST Storefront API. The Theme Engine will integrate with the Volusion Storefront API to support limitless possibilities. All APIs are RESTful with a detailed documentation reference and mock system for testing your ideas towards customizations. Turn your designer concepts into responsive realities for your storefront.

You can start following our progress and work with our available sample store and theme! Updates coming as we have them!

What is a responsive theme?

Mix design concepts, CSS styles, JavaScript, custom code, and web components and images into a theme that changes with any viewing device...and you have responsive design. The site interacts with shoppers on desktop browsers, tablets, and mobile devices seamlessly to browse, search, and purchase your products. Beauty and function.

How it works!

The Volusion Theme Engine powers your storefront themes. The themes and engine make calls to the Storefront API for retrieving data, creating and updating objects, and enhancing shopper experiences. The API is a fully RESTful API, providing HTTP verb operations for GET, PUT, POST, and DELETE through endpoints. Every request receives JSON. Every theme and app provides the interactions.

How do I use this guide?

This documentation grows with our features and your needs. We want to support your goals.

Designers may want to customize styles of text, images, colors, and slider images. Developers may want to deeply change the core of the themes or add custom apps to interact with the APIs and data. Your store. Your design.

The top navbar gives you options for installing tools, customizing themes, visiting Apiary, and contacting us!
