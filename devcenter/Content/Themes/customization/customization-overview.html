Customizing Your Theme

You can begin creating a theme for the theme engine from scratch, or you can get a copy of a seed project from the Volusion Theme Engine GitHub. For details on installing the theme, see Download a Volusion Theme.

These docs use the Method theme for examples. The Method theme can be forked or cloned to your GitHub client. Once you have a copy, you can sync and update directly through the GitHub client.

Customize Your Theme

You have the tools and a seed theme. But what should you do first? This list can point you in the right direction!

Options What You Can Do
The Quick and the Awesome: Quick start look at the best files and practices to follow when adding components and modifying styles.
Folders and Files: Every Volusion seed theme includes folders and files does following the same structure. You can find everything yo need in these folders!
Storefront Style: Dig into the Sass styles available. Learn more about BEM, SCSS, and how to convert styles you already have in CSS. Also see what tools are available for editing these files.
Image Sprites: Create sprite maps of icons and images using Sass and Compass.
Fantastic Fonts: You can change the color and weight of text, but why not change the very text itself?
Custom Text: Add custom text per pad or additional language content for supporting international shoppers.
APIs Made Easy: So you have styles and amazing images. Now let's get creative with code. See how the seed themes make API calls to the Storefront API!
