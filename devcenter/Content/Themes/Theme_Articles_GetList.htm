Get a List of Articles

This GET method requests a full list of articles from your store. The response will provide a list of articles including the ID, title, body and SEO settings. Depending on the amount of articles you create and use in your store, the response may be lengthy with content.

Request

The URI call structure is:

GET http://{domain}/api/v1/articles/id

Parameters

The following parameter can be entered into the request:

Parameter Description Type
id Alphanumeric system ID for the specific article. Number

Response

The response includes a response code and data for the all categories of articles in the store.

Status Code 200

JSON Response 
{
	"cursor": {
		"total": null,
		"totalPages": null
		},
	"data": {
		"id": 1,
		"title": "Company History"
		"body": "Company Name - Company Slogan - History of company, mission statement, etc.",
		"seo": {
			"metaTagTitle": null,
			"metaTagKeywords": null,
			"metaTagDescription": null,
			"metaTagsOverride": null
		},
	}
}

Data

The response includes a collection of articles available to the storefront.

The following cursor data is returned to set the pagination of response product data:

Attribute Description Type
total Numeric amount of all categories returned matching the request. Number
totalPages Numeric amount of total pages for pagination of the returned categories. The total amount is divided across the total amount of pages. Number

The following data is returned for the articles, with nested attributes for the articles returned:

Attribute Description Type
id The generated ID for the article when created. This is generated automatically by the Volusion system. Example: 1 Number
title The title of the article entered at article creation. The name is alphanumeric and supports some special characters. This is a required setting and always returns content. Example: Company History. String
body This information is the heart of the article, including all content entered. The content can include plain text or HTML formatted information, including links and images.

String
seo

This data sets the SEO friendly content for the article, including the following:

  • metaTagTitle - Sets an SEO friendly name for the category. If NULL, the data defaults to the category name.
  • metaTagKeywords - Sets all keywords used by search engines to locate and return this category in search results. Keywords are separated by commas. This is an unlimited String.
  • metaTagDescriptions - Sets the category description used in search engine results. This is a 255 character String.
  • metaTagsOverride - Sets advanced code that overrides all existing HTML meta tag information. This option is for advanced users.
 String
