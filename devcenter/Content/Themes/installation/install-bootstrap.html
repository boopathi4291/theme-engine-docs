About Bootstrap

Bootstrap provides a feature packed, versatile framework for developing web sites and apps. Bootstrap comes with layouts, formats, and styles you can extend and customize for immediate formatting. The framework supports your Sass CSS, custom JavaScript, and components like jQuery and Custom Elements.

If you develop styles using Bootstrap, you can even use Bootstrap themes to quickly shift and apply new looks to your entire site developed by others, such as those found at Bootswatch.

You have a few options for downloading and installing Bootstrap. Each install differs depending on your skill level. Check out the following options!

Easy Bootstrap

If you are not a strong developer, don't worry! You can add the Bootstrap CSS file and JavaScript file to your theme code to use layouts, formats, and modules available by importing the CDN styles. You can incorporate them into your styles and designs, previewing through a browser. This is an easy way if you just want to edit styles.

You will need to open your main CSS or SCSS file to add the CDN CSS for Bootstrap using @import:

@import url('//maxcdn.bootstrapcdn.com/bootstrap/3.2.0/css/bootstrap.min.css');

Additionally, you will also want to add the following JavaScript to your theme files <head> tag:

<script src="//maxcdn.bootstrapcdn.com/bootstrap/3.2.0/js/bootstrap.min.js'"></script>

For the full list of links, visit the Bootstrap CDN.

Download Bootstrap

You can download and install Bootstrap in one of three ways. For full details on what is included, see the Bootstrap Download page.
