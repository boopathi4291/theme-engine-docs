Theme Settings

Each theme can now deliver unique, data-driven functionality like never before! An example of innovative design is the home page slider, which allows merchants to create pr9omotinal banners for their storefronts. Othe examples includes social media widgets, promotional content, and much more.

Language and Custom Text

Ecommerce has gone global, requiring support for multiple languages and custom textual content. The text content files give you the power to develop language support, including cutomizing the content displayed per page, per section, right down to the options and fields in the storefront theme. Each file is saved and associated by country code.

CSS Overrides

Every theme can include a set of CSS overriding styles, customizable through the Theme Settings page or as entered by merchants when they install and activate your designed theme. These settings save to an override file, retaining the original styles and extending with custom ones. Merchants can easily changed or remove these settings to use the original styles you code into your theme.
