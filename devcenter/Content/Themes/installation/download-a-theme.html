Download a Volusion Theme

We designed the Theme Engine with three main goals: to give you unlimited creatity, with incredible features built-in, and access to a powerful API to build that site you've been dreaming of.

Volusion provides a set of themes from a simple to a deeply features version for downloading, customizing, and deploying for your storefront. The theme choices and options will continue to grow, including a theme storefront for purchasing and using themes! The following Volusion themes are available for download and developing for your storefront. Click the live demo link to preview or GitHub to download.

Installation Notes

A list of theme repositories displays to pick and choose from. You can fork, branch, or clone the code through the GitHub page. If you are savvy with Git commands, you can use a command prompt with the following command:

git clone git://github.com/username/repo.git

Everytime you download a theme, you need to issue the following commands via a command prompt/terminal program for both Windows and Mac. The commands sets grunt serve to run against your local copy, show your changes, and ensure you have all dependencies.

  1. Open a command prompt.
  2. CD move to the folder location of your GitHub theme code, and enter the following commands.
npm install
bower install
