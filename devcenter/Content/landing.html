Modern Stack

Using the latest front end development frameworks

AngularJS
AngularJS gives you an extensible environment to enhance HTML to fulfill all of your app and theme development dreams. Design dynamic views, extend HTML to fit your needs, integrate favorite libraries all within this robust toolset and framework. Volusion develops Theme Engine themes and projects on AngularJS.
SASS
CSS superpowered, Sass can integrate with your CSS and provides features and capabilities to design incredible styles and layouts. You can even integrate your Bootstrapped work within Sass and work in numerous frameworks, including Compass. Volusion actively uses Sass and Compass to develop theme styles.
API
HTML and CSS are just the start of your theme development adventures. Volusion Theme Engine is a powerful set of public, RESTful APIs for interacting with your storefront and shoppers, creating themes and custom apps, and designing advanced features such as faceted search and responsive layouts for a complete experience.
