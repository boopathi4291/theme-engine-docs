Welcome to Volusion Themes

Storefronts and company websites have become far more interactive experiences for shoppers, accessed on a variety of devices, mobile phones, and computers. Volusion sees the trends and has incorporated new opportunities to match your business with the latest technologies. You can add new responsive themes to your storefronts, powered by a theme engine and supported with a REST Storefront API.

There are no limits to what you can create with the Theme Engine. Let your imagination run wild and create that site you have been dreaming about. The Theme Engine includes a variety of pre-built starter themes to empower you to create beautiful websites with incredible capabilities. You can leverage any number of pre-built components or code and use your own web components with these designs.

The Theme Engine works with the Volusion Storefront API to support limitless possibilities. All APIs are RESTful with a detailed documentation reference and mock system for testing your ideas towards customizations. Turn your designer concepts into responsive realities for your storefront.

What is a responsive theme?

It is a series of design concepts, CSS styles, JavaScript, custom code, and additional web components and images packaged into a theme to completely style the look, feel, and interactivity of your storefront. The design responsively displays based on the desktop browser, tablet, or mobile device loading the storefront, giving shoppers a seamless experience to browse, search, and purchase your products. As shoppers visit your site, they receive a similar storefront without confusion or constant adjusting of the screen.

How it works!

The Volusion Theme Engine is a tool used to create customized web designs as themes for your storefront. The themes and engine make calls to the Storefront API for retrieving data, creating and updating objects, and enhancing shopper experiences. The API is a fully RESTful API, providing HTTP verb operations for GET, PUT, POST, and DELETE through endpoints. Every request, entered using a specific request envelop, receives a response including a status code and JSON as appropriate. The themes display the content retrieved using the customized styles and components.

How do I use this guide?

This documentation provides overviews and procedures for Web Designers and Web Developers. You can interact, design, and code to any level you are comfortable with using this content. As a designer, you may only want to changes the styles of text, images, colors, and slider information. As a developer, you may want to deeply change the core of the themes or add custom apps to interact with the APIs and data. Your store. Your design.
