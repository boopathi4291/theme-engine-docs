Adding and Using Fonts

Quickly add a new font to your theme.

Typography is one of the most widely used content and important pieces in any webpage and site. Fonts can provide a facelift, increasing legible reading online and for print of your text. You can use incredible fonts and text not only to provide descriptions and headings, but also lends to logo design, product images, and creative backgrounds.

Each theme seed project includes a number of fonts in the {your repo}\method\app\fonts\ folder. For responsive themes, you should add one of all four font types to support all devices, desktops, mobiles, and browsers.

All About Fonts

Font types include:

  • Embedded Open Type (.eot) - These are compact OpenType fonts that can be created from TrueType fonts. You can create these as needed using Microsoft's Web Embedding Fonts Tool. These are typically loaded for Interest Explorer older than IE9.
  • Scalable Vector Graphic (.svg) - An XML-based vector image format for 2D graphics. These can be created with text editors, but typically designers create them with graphic apps.
  • TrueType (.ttf) - One of the most common font types available, usable on Mac and Windows systems and most device browsers.
  • Web Open Font Format (.woff) - These are OpenType or TrueType fonts with compression and metadata, powerfully helpful for bandwidth usage. This is a widely popular font and supported by iOS 5.

Method provides the following fonts in the GitHub project: Antonio (bold, light, regular), Crimson Text (regular, italic), GLYPHICONS Halflings (regular), Roboto (regular, bold, bold italic, italic, condensed regular, condensed bold).

When adding a font, you can purchase font files, create your own, or use the sets already provided in the theme seed projects. You can also use font generators found online. If you have a few fonts with only a couple of formats, you may find a font generator (to provide the other font formats you may need. One example of a site is @font-face Generator.

Having all four font formats is recommended, though you may get away with only supporting a few key formats.

Fonts as Buttons

Numerous fonts are available for use as buttons. Using fonts as an icon for a button provides a range of options as vectors, resizable without pixelating! The best and most supported with Bootstrap include glyphicons, Font Awesome, IcoMoon, Ionicons, and Octicons. You can use Glyphsearch to find the right font image for you.

One of the best is Font Awesome, with a full range of icon images including social icons! If you @import the Font Awesome SSC from their CDN link, you can add these icons as buttons. Here is an example of a Facebook button!

The code for this includes:

<a href="https://www.facebook.com/Volusion"><button type="button" class="btn btn-primary"><i class="fa fa-facebook"></i></button></a>

Adding Font Files

The new responsive themes continue to develop as Volusion works on all the incredible options available within storefronts and your store data. As features progress, what pages are available will change to help you go further with every customization, cool idea, and amazing new app. You have a couple of methods for adding a new font to your theme. The following instructions detail one method that includes your font files to your theme repository, font list, and variables. This way, you only have to add the files, the name to the font list, and add the font to any styles via variables without searching out every single style.

Step What to Do
1 Open the fonts folder: {your repo}\method\app\fonts\.
2 Add your font files to this folder and the repository. You should include every file type with the font (.ttf, .eot, .woff, .svg). Make sure every font file name is named to reflect the name you want to call the font in your stylesheet. For example, font-family: Roboto-Italic; will need to have all file names changed to "Roboto-Italic" for all file types. Every new .svg file will need to have its ID changed to "font".
3 With the fonts added, you need to add an @include line to the _fonts.scss stylesheet extension. Locate the file at {your repo}\method\app\styles\base\_fonts.scss.
4 Open this file and add declarations for your new font. For each new font, add an @include app-font('font-name-here'); line: @include app-font('roboto-regular');.
5

You can then add the font to a specific font family in the _bootstrap-variables.scss stylesheet extension located at {your repo}\method\app\styles\base\_bootstrap-variables.scss. The typography section includes variables for font families to be used throughout your theme's styles. For example, you can add "robotoregular" to the font-family-sans-serif. This becomes a default font to use for this family in styles and web browsers: $font-family-sans-serif: "roboto-regular", Arial, sans-serif !default;.

Adding Google Fonts

Google also provides fantastic fonts to use in your themes. You can review and test out all of their fonts at https://www.google.com/fonts. When you have found a font you like, do the following to add it to your theme using an @import line of code. The standard and Javascript options are directly added to the code of your theme pages and templates rather than handled in your styles like other fonts. You can use any of these options.

Step What to Do
1

Click the Quick-use button for the font you like.

2 Select the styles you want for the font. You can select one or more using the checkboxes. Each option has a sample.
3 Select the character sets, one or more. Latin is the default for the font. Remember, you can always add more later!
4

You then have a choice of standard, a generated @import line, and Javascript. We use the @import for ease of use and simplicity. Copy that line of code, such as: @import url(http://fonts.googleapis.com/css?family=Open+Sans);
5 Time to add the font to your styles. Locate and open the _fonts.scss file at {your repo}\method\app\styles\base\_fonts.scss.Add the @import line provided by Google: @import url(http://fonts.googleapis.com/css?family=Ubuntu);.
6

You can then add the font to a specific font family in the _bootstrap-variables.scss stylesheet extension located at {your repo}\method\app\styles\base\_bootstrap-variables.scss. The typography section includes variables for font families to be used throughout your theme's styles. Ubuntu is a sans serif font that could be added to that this line:$font-family-sans-serif: "Ubuntu", "roboto-regular", Helvetica, Arial, sans-serif !default;.
