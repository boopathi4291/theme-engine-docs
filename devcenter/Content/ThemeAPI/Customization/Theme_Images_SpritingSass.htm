Sass Sprite Maps

You can create any size of image to use as a sprite, as a jpg, svg, or png, for use in your theme. Maps give you the ability to add multiple images all matching in dimensions into a single image for use within SCSS files to insert into your page templates.

If you want to automatically generate sprite maps and CSS code, consider using Compass!

The following image is a sprite map image of common formatting icons for text, images, and attachments. Each icon image is could be 24px by 24px, which becomes the size of each tile. This image would then have 8 tiles across and two tiles down.

The SCSS code would then look like the following for a media icon (used for inserting or loading a video):

$icon-width: 24px;
$icon-height: 24px;
$icons: image-url('toolbar.png');

.media-icon {
	background-image: $icons;
	background-position: -($icon-width * 5) -($icon-width * 1);
	width: $icon-width;
	height: $icon-height;
}

This code would load an icon image located five tiles over and one tile down, 24px per tile. That image is highlighted below:

When the media-icon style is used, that icon displays!
