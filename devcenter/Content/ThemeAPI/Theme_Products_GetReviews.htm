Get Product Reviews

This GET method retrieves the available reviews for a specific product. The method allows you to enter a specific product code to retrieve shopper entered review comments and ratings. These reviews in turn could be highlighted with design, page layouts, and additional app functionality for enhancing shopper purchases. The resulting data includes numerical ratings, a number of total reviews for the product, and all specific reviews by poster, rating, content written, the number of shoppers finding it as helpful, and posted date.

You can review these entered shopper comments and rating through the Admin Customer > Customer Reviews page. Shoppers can enter their name, geographic location, a title for the review, comments, and a star rating. Each star or half star is has an equivalent numerical value based on the minimum and maximum values sets for reviews.

The response returns a collection of review data. Depending on the entered responses by your shoppers, the response could be lengthy.

If you have reviews set to review prior to publishing or an inactive setting turned on for a review, these reviews may not be returned for the request. Hidden products may not return when queried from the store.

Request

The URI call structure is:

GET http://{domain}/api/v1/products/code/reviews

Parameters

The following parameter can be entered into the request:

Parameter Description Type
code Alphanumeric product code for the product. All product reviews are associated to the product via this code. Example: mermaid-tshirt-summer-2014. Number

Response

The response JSON provides a status code 200 and data for the specific product on success.

Data

The response includes a section of JSON detailing the cumulative results of the reviews and a collection of all entered reviews for the specified product code.

The following cursor data is returned to set the pagination of response product data:

Attribute Description Type
total Numeric amount of all product reviews returned matching the product code. Number

The following data is returned in a collection, one data element per product for the request:

Attribute Description Type
cumulativeRating The averaged score for the product based on all entered numerical ratings by shoppers. The amount will fall within the minimum and maximum rating allowances. Number
minRating The minimum rating allowed for a product review. This value is zero by default. Number
maxRating The maximum rating allowed for a product review. This value is five by default. Number
numberOfReviews The total amount of entered reviews for the product code.

Number
reviews

This is a collection of review objects, providing all data for the shopper entered review. Not all reviews may include a title or comment.

  • rating - The numerical rating assigned to the product by the shopper. This rating is associated to the number of stars selected for the product. The cumulativeRating averages this score for all review objects.
  • nameOfReviewer - Name of the shopper reviewing the product as entered by the shopper. This is not pulled from a logged in account name for the shopper.
  • title - A title entered for the review.
  • body - The full comment content entered by the shopper.
  • numberOfPeopleWhoFoundReviewHelpful - A numerical amount of other shoppers that selected this review as helpful when considering a purchase of the product.
  • date - System generated date of the entered comment.

String

Number
