Get Logged In User

This GET method requests the information for the currently logged in user account on the storefront. The response includes the user account ID, full name, and email address. You can use this content to determine who is logged in, and use that ID to gain further user account information or provide additional services per the logged in user.

Request

The URI call structure is:

GET http://{domain}/api/v1/users/current

Parameters

This request does not require parameters.

Response

The response includes a response code and data for the logged in user.

Status Code 200

JSON Response 
{
  "data": {
	"id": 9317
	"email": "johnsmith@gmail.com",
	"firstName": "John",
	"lastName": "Smith",
	"isAnonymous": false,
	"storeCreditAvailable": 0
	}
}

Data

The response includes data for the logged in user.

The following data is returned for the countries:

Attribute Description Type
Id The generated ID for the currently logged in user on the storefront. String
email The email address for the currently logged in user.

String
firstName The first name for the currently logged in user. String
lastName

The last name for the currently logged in user.

 String
isAnonymous Indicates if the user is anonymous. Boolean
storeCreditAvailable The amount of store credit available for the user account. This credit can be used towards purchases and displays as credit in the shopping cart during checkout. Number
