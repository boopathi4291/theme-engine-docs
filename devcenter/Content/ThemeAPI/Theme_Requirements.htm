Requirements

The following information guides you through the information, tools and skills you will need for working with the Volusion Storefront API.

Tools and Technology

The REST API accepts requests from apps, mobile devices, desktop implementations and any other creative code you can dream up. The Storefront API was created using the following tools:

  • Twitter Bootstrap (Sass) - Designed to make development easier and efficient. Bootstrap is fully responsive from the mobile device to the desktop.
  • Sass and Compass - The popular CSS-extension language, Sass, powers Compass to allow for code that’s DRY, readable and easy to maintain.
  • BEM CSS Syntax - BEM (block, element, modifier) is a methodology for creating code that’s easy to read, maintain, and reuse across themes.
  • AngularJS - AngularJS is an open-sourced framework used to create single-page web applications (SPAs). While it is not required to know AngularJS when developing a new Volusion theme, it is recommended if you plan on extending an existing theme's functionality or choose to develop new components.
  • Volusion Storefront API - The core set of REST APIs for building unique storefront experiences for your themes and additional apps to empower them.

You should have your tools installed and environment configured to continue. If not, see Prepping Your Environment.

Theme Code

You can download a copy of the Volusion Themes from Git Hub. These Git repositories are all-inclusive seed projects for building upon for your own themes. All Volusion responsive themes are built on the AngularJS framework. For the basis of examples in this documentation, you will follow code and samples from the Method theme.

Required Skills and Knowledge

Developing themes does require an understanding of the following technologies:

  • JSON - A solid understanding of extending and using REST API endpoints, particularly using JSON implementations is required. You will interact with the endpoints, sending requests and receiving data in JSON.
  • HTML - HTML 5 is not required but recommended. Many of the base pages and templates may use HTML, requiring tweaking to edit and add content and formatting.
  • CSS - A solid understanding of CSS and how CSS preprocessors work is required. Styles provide a depth of design control over all aspects of your theme from a single source.
  • JavaScript - JavaScript knowledge is required for theme features and widgets, but not for layouts. Basic JavaScript knowledge helps with minor modifications, but if you plan to redo basic features built into a current theme you need solid JavaScript skills.
  • Graphic Editing - Optionally, you may need an understanding of graphic editing and creation for any images you want to add to your designs or for working with large graphics for bold sliders and backgrounds.
