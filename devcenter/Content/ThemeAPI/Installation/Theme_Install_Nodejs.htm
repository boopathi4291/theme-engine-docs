Step 2: Install Node.js

If you have Node.js installed, continue to step 3.

Node.js is the JavaScript framework used to develop and support Volusion Themes. The framework is an event driven, non-blocking I/O model for lightweight and efficient development for scalable network programming. Node is similar to Ruby's Event Machine and Python's Twisted for the seasoned web developer, while extending the event loop itself as a language construct rather than a typical library.

For deeper information on Node.js and their API reference, you can review their documentation and even fork their GitHub repository.

Download Node.js

Node.js provides installations for Windows (32 and 64 bit) and Mac OS X. Visit their website and click the large INSTALL button. An automatic version will download for your system. Or you can select a version from their downloads page. Additional installations are available on the page, but you are only required to install one version to support Volusion Theme development.

Installation Notes

You can follow the prompts when installing the framework via the executable you downloaded. You do not need to create a clone or fork their GitHub for development.

Windows Installation

The Windows installation uses a wizard.

Step What to Do
1 Open the installation file. An installation wizard displays to walk you through the process.

2 Click Next.
3 Read through the end-user license agreement and click the acceptance checkbox.

4 Click Next.
5 Select a specific destination folder for the install, or use the default location provided.

6 Click Next.
7 You can customize your setup on the next screen, selecting specific features, reviewing the hard drive space required. For this installation, you can continue as is with all features installed.

8 Click Next.
9 You are ready to finish installation! Click Install.

10 A progress screen displays. If installing on Windows, you may be prompted to allow the installation.
11 Click Finish to complete.
Complete!
Mac Installation

These instructions are completed through command line, and take only minutes.

Step What to Do
1 Open a Terminal application.
2 First, let's check if you have Node.js installed. Type and enter: node --version If a version displays, you are done.
3 If you do not have Node.js, visit the website and click INSTALL.
4 Run the installation, following the screens without changing settings.
5 Once installed, you can test again. Type and enter: node --version If a version displays, you are done.
Complete!
