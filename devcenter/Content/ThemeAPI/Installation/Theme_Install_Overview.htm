Getting Started

Before you can create new themes, or edit current theme seed projects, you need to install a few tools and set up your environment. This process does not take long, and will give you the power to not only work with the Volusion seed theme projects, but develop your own powerful apps and custom responsive themes!

Let's get started!

Install and prepare development tools and environment locally.
Select and get a theme repository.
Customize your theme.
Upload your theme.
Test your theme.
Update your store in the Admin to run this theme.
Deploy to the world!

Getting the Theme Code

With all tools installed, you can now select and download a theme. All theme code is available at the Volusion Theme GitHub. You can use your GitHub client to access the code or use a command prompt to create a clone.

If you plan on contributing,you may want to fork the repository rather than create a clone. You can find instructions for forking a repository at http://help.github.com/fork-a-repo/. After forking the repository, you will want to create a local copy of your fork.

Additional Helpful Information

Have questions or need help? Volusion provides places to post your questions and review previous answers from our developer community!

  • Dev Community @ Stackoverflow
  • Volusion Dev @ User Voice

You can make direct API calls to the Volusion Storefront API through theme code, apps, widgets, and more. You can view this guide and the Storefront API on Apiary documentation with mock test console!

The Volusion Themes are built on the Angular.js framework. You can review the developer guide for deeper details and information regarding the supported functions and features through this framework.
