Delete a Cart

The DELETEmethod is used when deleting the entire cart object, best used when the order has been placed and enters the processing stage (post-checkout). You do not want to delete the cart prior to final payment submission as the shopper may want to change a quantity for a product or add another product to the cart during the checkout process. The response only provides a 204 status code on success.

Request

The URI call structure is:

DELETE http://volusion.apiary-mock.com/api/v1/carts

The header is: vMethod: DELETE.

Parameters

The cart ID is not required for the request as a cookie provides any required cart identification. This method does not require parameters.

Response

The response includes only the status code for success or failure.

Status Code 204
