Add to Theme Marketplace

The Markplace is not approved. Will continue this content when the Marketplace is approved.

With your theme prepared for selling, you can add the theme to the Marketplace. You will need the following:

  • The GitHub URL and version for the theme files. You will set this information through GitHub when managing the theme files.
  • Images of the theme in use. (img requirements?)
  • Features and information for the theme.

To add your theme:

  1. Enter the Name for the theme. This is the name that merchant will see and select when purchasing the theme in the Marketplace.
  2. Enter a Description. Provide details on what makes your theme special, supported features, design concepts, and type of store best suited to use it.
  3. Enter a list of Features. You can use HTML formatting to breakout content into lists.
  4. Upload Images of your theme in use. You could include a full image of the main landing page, a category, a product, and images of the theme across browsers and devices. Once uploaded, the images are added to a list with a URL, generated Id, the order they will display in, and options to delete and reorder the files. The images display in a carousel for the theme, for potential users to review prior to purchase.
  5. Select a Color for your theme.
  6. Select if the theme is a Premium theme. If this is unchecked, the theme is considered free. If checked, the theme receives premium pricing for purchase and use by Volusion Merchants.
  7. Enter the GitHub URL for the theme files.
  8. Enter the Current Version. This if the version for the URL linked file. You should keep the link and version in sync.
  9. Enter the ID for the Matching Legacy Theme Id. If a merchant uses features not supported in your theme, this legacy theme will be used for those pages and actions.
  10. Copy and paste any CSS code you want to use for the checkout in the Checkout CSS Override. These styles can be used to match the legacy theme checkout process pages to your theme.
  11. Enter your name or the designer's name for Created By.
  12. Enter a URL to the Logo/Headshot of the designer. This could be a logo design or a picture of the designer.
  13. Enter the URL to the designer Website.
  14. Enter a title, such as the job Title for the designer.
  15. Enter the Company name.
  16. Enter a Short Description about the designer or company providing this theme. This is a great chance to sell your company and work.
  17. Enter a Support Email for contacting the designer or company with questions regarding the theme.
  18. Enter a Dribbble Handle.  This is a handle for Dribbble, a showcase website for designers.
  19. Click Save.
